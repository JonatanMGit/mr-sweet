## Installation

### Bot

To install the bot:

1. Clone the repository
2. Navigate to the `bot` folder
3. Run `npm install`
4. Rename `.env.example` to `.env` and fill in the values
5. Navigate to the `build` folder
6. Run `npm run build`
7. Start the bot by running `npm start`
